A lightweight web-based examination platform built with Flask that allows teachers to create exams with embedded diagrams and interactive drawing areas. Students can answer questions, annotate diagrams using digital drawing tools (pencil, line, ray, eraser, undo/redo), and save their work with automatic overlay capture. Teachers can review attempts, view diagram annotations, assign marks, and provide feedback through a structured marking interface. The system is designed for easy deployment, minimal dependencies, and seamless integration with LMS platforms such as Moodle.
